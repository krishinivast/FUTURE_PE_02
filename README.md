## 🧠 Ad Framework Used

All scripts follow the proven UGC conversion framework:

```
HOOK (0–3s)
  ↓ Stop the scroll
PROBLEM (3–10s)
  ↓ Agitate the pain point
SOLUTION (10–Xs)
  ↓ Introduce the product naturally
PROOF (X–Ys)
  ↓ Show a real result
CTA (last 5s)
  ↓ One clear action
```

## 🔄 How to Reuse for Any Brand

1. Open `prompts_framework.md`
2. Copy **Prompt 5 (Brand Adaptor)**
3. Paste your existing script + new brand details
4. Run in ChatGPT / Claude / Gemini
5. Lightly edit output for authenticity
6. Deliver to client ✅

**Time per new brand:** ~10 minutes
