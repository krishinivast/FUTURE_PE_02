# ⚙️ UGC Ad Prompts Framework

> A reusable prompt system for generating high-converting UGC ad content.  
> Works with: **ChatGPT · Claude · Google Gemini**  
> Swap `[VARIABLES]` for any brand, product, or niche.

---

## How to Use This Framework

1. Copy any prompt below
2. Replace all `[VARIABLES]` with your client's details
3. Paste into ChatGPT, Claude, or Gemini
4. Review output → lightly edit for authenticity
5. Deliver to client or use for your own content

> ⚡ **Pro tip:** Use Prompt 4 (Brand Adaptor) to repurpose any existing script for a new client in under 5 minutes.

## Prompt 1 — Hook Generator

**Use when:** You need scroll-stopping opening lines for a new product.

```
You are a UGC ad copywriter for D2C brands.

Generate 7 scroll-stopping HOOKS for this product:
- Product: [PRODUCT NAME]
- Core problem it solves: [PAIN POINT]
- Target audience: [AUDIENCE DESCRIPTION]
- Price: [PRICE]

Rules for hooks:
→ Sound like a real person talking, NOT an ad
→ Use one of these types: Pain Point / POV / Before-After / Curiosity / Authority / Honest Review / Social Proof
→ Max 2 sentences. First sentence stops the scroll. Second earns the watch.
→ Zero corporate language. Zero buzzwords like "game-changer" or "revolutionary".
→ Label each hook with its TYPE and explain in one line WHY it works.
```

**Expected output:** 7 hooks with type labels and rationale.

## Prompt 2 — Full Ad Script Generator

**Use when:** You need a complete, ready-to-shoot UGC video script.

```
Write a UGC-style video ad script for the following product.

Product details:
- Product name: [PRODUCT NAME]
- Platform: [Instagram Reels / TikTok / YouTube Shorts]
- Target duration: [15s / 30s / 60s]
- Tone/style: [honest review / before-after / skeptic-turned-believer / expert tip / day-in-my-life]
- Key result or USP: [THE MAIN RESULT THE PRODUCT DELIVERS]
- Offer and promo code: [OFFER DETAILS + CODE]

Structure the script STRICTLY in this order:
1. HOOK (0–3s): Stop the scroll immediately
2. PROBLEM (3–10s): Identify and agitate the pain point
3. SOLUTION (10–Xs): Introduce the product naturally — not salesy
4. PROOF (X–Ys): One real result, visual moment, or testimonial line
5. CTA (last 5s): ONE clear action only — no multiple asks

Critical rules:
→ NEVER say: "I partnered with", "sponsored by", "use my affiliate link"
→ Sound exactly like a real customer sharing a genuine find with a friend
→ Include brief stage directions in [brackets] for camera/action cues
→ Timestamp each section (e.g. "[HOOK — 0 to 3s]")
```

**Expected output:** Complete timestamped script with stage directions.

## Prompt 3 — Caption + Hashtag Generator

**Use when:** You need post copy to accompany the video ad.

```
Write a social media caption for a UGC ad post.

Details:
- Product: [PRODUCT NAME]
- Hook line used in the video: [PASTE HOOK HERE]
- Platform: [Instagram / TikTok / YouTube Shorts]
- Caption style: [storytelling / short-punchy / educational / minimalist]
- CTA to include: [CTA TEXT]
- Promo code: [CODE]

Deliverables:
→ Caption body:
   - TikTok: 2–5 lines max
   - Instagram: up to 8 lines, with line breaks for readability
   - YouTube: 4–6 lines + link mention
→ 8–12 hashtags (mix of niche + broad reach)
→ 1–2 relevant emojis max — only where they feel natural
→ Tone: raw, real, non-corporate — like a friend posting, not a brand
```

**Expected output:** Ready-to-paste caption + hashtag block.

## Prompt 4 — CTA Variation Generator

**Use when:** You need multiple CTA options for A/B testing.

```
Generate 8 different CTA (call-to-action) endings for this UGC ad.

Product: [PRODUCT NAME]
Platform mix: [LIST PLATFORMS]
Promo code: [CODE]
Key trust element: [e.g. "21-day refund", "dermatologist recommended", "5,000 reviews"]

Generate one CTA for each of these strategies:
1. Casual / friend-like
2. Short and punchy (for TikTok)
3. Trust-building (leans on guarantee)
4. DM-driven (asks them to DM a keyword)
5. Urgency / scarcity (feels like a tip, not pressure)
6. Offer-led (bonus or discount)
7. Skeptic softener (removes risk objection)
8. Community loop (encourages buyer UGC)

Each CTA: max 2 sentences. No exclamation marks overload. Sound human.
Label each with its strategy type.
```

**Expected output:** 8 CTA variants with strategy labels.

## Prompt 5 — Brand Adaptor (Reuse for any new client)

**Use when:** You want to repurpose an existing script for a different brand quickly.

```
Adapt the following UGC ad script for a completely different brand.

Original script:
[PASTE EXISTING SCRIPT HERE]

New brand details:
- Brand name: [BRAND NAME]
- Product category: [CATEGORY — e.g. fitness supplement, hair care, SaaS tool]
- Target audience: [NEW AUDIENCE]
- Core USP: [WHAT MAKES THIS PRODUCT DIFFERENT]
- Tone: [casual / professional / humorous / emotional / authoritative]
- Promo code: [CODE]

Instructions:
→ Rewrite completely — do NOT mention the original brand or product
→ Preserve the exact same structure: Hook → Problem → Solution → Proof → CTA
→ Match the new tone and audience
→ The output must feel 100% original, not like a template swap
→ Include updated stage directions in [brackets]
```

**Expected output:** Fully adapted script ready for the new brand.

## Prompt 6 — Batch Content Calendar

**Use when:** A client needs a month of UGC ad variations.

```
Create a 4-week UGC ad content calendar for this brand.

Brand: [BRAND NAME]
Product: [PRODUCT NAME]
Platforms: [PLATFORMS]
Monthly theme: [e.g. "Summer Glow", "Back to Routine", "New Year Skin Reset"]

For each week, generate:
- 1 primary ad script (full length for the main platform)
- 2 short hook variations to test
- 1 post caption with hashtags
- 1 CTA variant

Week themes:
- Week 1: Introduce the problem
- Week 2: Introduce the solution
- Week 3: Social proof / results
- Week 4: Urgency / last chance offer

Keep all content feeling like different real customers — vary the tone, hook type, and perspective each week.
```

**Expected output:** 4-week content calendar with scripts, hooks, captions, and CTAs.

## Variable Reference Guide

Replace these placeholders across all prompts:

| Variable | What to Put Here |
|---|---|
| `[PRODUCT NAME]` | Full product name (e.g. "GlowRx Vitamin C Serum") |
| `[PAIN POINT]` | The #1 problem the product solves (be specific) |
| `[AUDIENCE DESCRIPTION]` | Age, gender, lifestyle, specific frustration |
| `[PRICE]` | Actual retail price |
| `[PLATFORM]` | Instagram Reels / TikTok / YouTube Shorts |
| `[DURATION]` | 15s / 30s / 60s |
| `[TONE/STYLE]` | honest review / before-after / skeptic / expert tip |
| `[UNIQUE RESULT]` | The specific, measurable result (e.g. "visibly brighter skin in 21 days") |
| `[OFFER + CODE]` | e.g. "20% off with code GLOW20 + free shipping" |
| `[CTA TEXT]` | The exact action you want viewers to take |
